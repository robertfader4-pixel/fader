Итальянское настроение

Sube el contenido manteniendo exactamente esta estructura:
- index.html
- css/styles.css
- js/tracks.js
- js/app.js
- assets/images/...
- assets/audio/...

Notas:
- El reproductor usa la carpeta assets/audio.
- La playlist se puede arrastrar en escritorio desde su cabecera.
- El audiospectrum turquesa está sobre la miniatura visual del track.
- No se usan dependencias externas. Todo funciona offline.
